# Uridim

Uridim is a terminal-native, project-aware observability and diagnostics environment for developers.

You launch Uridim inside a software project, and it detects the technologies used in that repository. Based on the detected stack, Uridim exposes only the commands, metrics, health checks, and live terminal widgets relevant to that project.

## Status

Uridim is currently in early development.

## Vision

Enter a project. Uridim understands the stack and shows how the project is actually behaving.
